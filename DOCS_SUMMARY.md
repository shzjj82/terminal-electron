## 🚀 使用方法

### 启动开发服务器
```bash
# 在 docs 目录下
cd docs
yarn install
yarn docs:dev

# 或在根目录下
yarn docs:dev
```

### 构建文档
```bash
# 构建静态文件
yarn docs:build

# 预览构建结果
yarn docs:preview
```

### 部署文档
```bash
# 构建文档
yarn docs:build

# 部署到 GitHub Pages
# 将 .vitepress/dist 目录的内容部署到 gh-pages 分支
```
